get /income/securities/{taxableEntityId}/{taxYear}

List of securities income

  • Request

URI Parameters

  • taxableEntityId: required(string)

    Unique identifier of the taxable entity to which the submission belongs i.e. NINO for an individual

    Example:

    TC663795B
  • taxYear: required(string)

    The tax year the calculation relates to, in the format YYYY-YY

    Example:

    2017-18
  • Response

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "securities": [
    {
      "type": "GILT-EDGED",

/obligations/{partyIdentifier} get

get

List obligations for the party

get /obligations/{partyIdentifier}

List obligations for the party

  • Request
  • Response

URI Parameters

  • partyIdentifier: required(string - pattern: ^([ACEHJLMOPRSWXY][A-CEGHJ-NPR-TW-Z]|B[A-CEHJ-NPR-TW-Z]|G[ACEGHJ-NPR-T-Z]|[KT][A-CEGHJ-MPR-TW-Z]|N[A-CEGHJL-NPR-SW-Z]|Z[A-CEGHJ-NPR-TW-Y])[0-9]{6}[A-D]?$)

    Party identifier, NINO for SA Individuals

    Example:

    TC663795B

Query Parameters

  • source: (one of selfEmployment, ukProperty, foreignProperty)

    The source context for the obligation e.g. self employment business

  • sourceId: (string)

    The associated source identifier for the obligation

  • type: required(one of incomeAndExpenses, selfAssessmentCrystallisation, vatReturn, selfAssessmentEndOfPeriodStatement)

    The type of obligation

  • status: (one of open, fulfilled)

    The type of obligation

  • start: (date-only)

    The start date to filter obligations by

  • end: (date-only)

    The end date to filter obligations by

Body

Media type: application/json

Type: object

Properties
  • code: required(string)

Example:

The format of the supplied NINO field is not valid

{
  "code": "FORMAT_NINO"
}

HTTP status code 403

Body

Media type: application/json

Type: object

Properties
  • code: required(string)

Example:

clientOrAgentNotAuthorised:
The client and/or agent is not authorised. This is normally because: the client is not subscribed to MTD, the agent is not subscribed to Agent Services, or the client has not authorised the agent to act on their behalf.

{
  "code": "CLIENT_OR_AGENT_NOT_AUTHORISED"
}

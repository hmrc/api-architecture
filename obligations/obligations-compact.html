get /obligations/{nino}/income-tax

List income tax obligations for the party

  • Request
  • Response

URI Parameters

  • nino: required(string - pattern: ^([ACEHJLMOPRSWXY][A-CEGHJ-NPR-TW-Z]|B[A-CEHJ-NPR-TW-Z]|G[ACEGHJ-NPR-T-Z]|[KT][A-CEGHJ-MPR-TW-Z]|N[A-CEGHJL-NPR-SW-Z]|Z[A-CEGHJ-NPR-TW-Y])[0-9]{6}[A-D]?$)

    National Insurance number, in the format AA999999A.

    Example:

    TC663795B

Query Parameters

  • start: (date-only)

    The start date to filter the results by

  • end: (date-only)

    The end date to filter the results by

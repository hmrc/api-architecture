/deductions/cis/{nino}/current-position get

get

List current position of CIS deductions.

get /deductions/cis/{nino}/current-position

List current position of CIS deductions.

  • Request
  • Response

URI Parameters

  • nino: required(base-types.nino)

    Unique identifier of the taxable entity to which the submission belongs i.e. NINO for an individual

Query Parameters

  • fromDate: required(date-only)

    Start date of the tax year in YYYY-MM-DD

    Example:

    2018-04-06
  • toDate: required(date-only)

    End date of the tax year in YYYY-MM-DD

    Example:

    2019-04-05
  • source: required(string - default: all)

    Source of CIS Deductions i.e. those supplied by the Contractor, the Customer or all (blended)

    Example:

    all

List current position of CIS deductions. [DES-1572]

get /deductions/cis/{nino}/current-position

List current position of CIS deductions. [DES-1572]

  • Request
  • Response

URI Parameters

  • nino: required(string - pattern: ^([ACEHJLMOPRSWXY][A-CEGHJ-NPR-TW-Z]|B[A-CEHJ-NPR-TW-Z]|G[ACEGHJ-NPR-T-Z]|[KT][A-CEGHJ-MPR-TW-Z]|N[A-CEGHJL-NPR-SW-Z]|Z[A-CEGHJ-NPR-TW-Y])[0-9]{6}[A-D]?$)

    National Insurance number, in the format AA999999A.

    Example:

    TC663795B

Query Parameters

  • fromDate: required(date-only)

    Start date of the tax year in YYYY-MM-DD

    Example:

    2018-04-06
  • toDate: required(date-only)

    End date of the tax year in YYYY-MM-DD

    Example:

    2019-04-05
  • source: (string)

    Source of CIS Deductions i.e. those supplied by the Contractor, the Customer; Implicit default is All

    Example:

    customer

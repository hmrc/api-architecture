https://api.service.hmrc.gov.uk

Assumptions

  1. Object schemas from backend [DES] API for self-employment businesses and properties are the same

Design Rationale

The data schemas for business and property are thought to be the same (or very similar), so it made sense to combine everything in one GET "list" and one GET "details". If we need to add further types of business details in future we can extend with finer-grained end-points as non-breaking changes.

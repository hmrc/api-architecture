/obligations/{partyIdentifier}/income-tax get

get /obligations/{partyIdentifier}/income-tax

List income tax obligations for the party

  • Request

URI Parameters

  • partyIdentifier: required(string - pattern: ^([ACEHJLMOPRSWXY][A-CEGHJ-NPR-TW-Z]|B[A-CEHJ-NPR-TW-Z]|G[ACEGHJ-NPR-T-Z]|[KT][A-CEGHJ-MPR-TW-Z]|N[A-CEGHJL-NPR-SW-Z]|Z[A-CEGHJ-NPR-TW-Y])[0-9]{6}[A-D]?$)

    Party identifier, NINO for SA Individuals

    Example:

    TC663795B

Query Parameters

  • type: (self-assessment-crystallisation)

    The type of obligation

  • status: (one of open, fulfilled)

    The type of obligation

  • start: (date-only)

    The start date to filter the results by

  • end: (date-only)

    The end date to filter the results by

  • Response

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "incomeTaxObligations": {
    "type": "self-assessment-crystallisation",
    "obligations": [
      {
        "start": "2017-04-06",
        "end": "2018-05-04",
        "due": "2019-01-31",
        "status": "open"
      }
    ]
  }
}
